MiraiMemberCardChangeEvent

interface MiraiMemberCardChangeEvent : MiraiGroupMemberEvent<MemberCardChangeEvent> , ChangedEvent(source)

成员群名片改动. 此事件广播前修改就已经完成.

由于服务器并不会告知名片变动, 此事件只能由 mirai 在发现变动时才广播. 不要依赖于这个事件.
