QGChannelCategory

interface QGChannelCategory : Category, QGNonTextChannel, QGChannelCategoryId(source)

频道分组。QGChannel 的实现类型之一。

当一个频道的 QGSourceChannel.type 的值等于 ChannelType.CATEGORY 时， 此频道代表为一个分组。

可以通过 QGGuild.categories 或 QGGuild.category 查询获取。
