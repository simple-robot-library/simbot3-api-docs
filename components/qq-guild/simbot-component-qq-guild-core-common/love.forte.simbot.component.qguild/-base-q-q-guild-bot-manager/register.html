注意，在配置 ConfigurableBotConfiguration 时， 如果 coroutineContext 中 存在自定义的 Job，那么 Application 中的Job 则会作为一个 Root Job 而不是 Parent Job 使用。

Root Job 仅会在其自身完成或关闭的时候通知相关联的子类使它们关闭， 但不会有硬性关联，这种通知是通过 Job.invokeOnCompletion 实现的， 参考 Job.registerRootJob。

如果 coroutineContext 中不存在自定义的Job，则会直接使用 love.forte.simbot.component.qguild.QQGuildBotManager 内的 Job 作为 parent Job 。
