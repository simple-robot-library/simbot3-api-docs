reply

abstract suspend override fun reply(message: Message): QGMessageReceipt(source)

基于当前事件进行消息回复。

对于公域bot来说，也理应这么回复。reply 会默认根据当前事件为当前可能产生的所有消息体填充回复的目标（msg_id 或 event_id）， 并且与 send 相比会默认追加一个 引用效果 （如果 message 中存在 自定义reference 则在其影响范围内不会自动填充）。
