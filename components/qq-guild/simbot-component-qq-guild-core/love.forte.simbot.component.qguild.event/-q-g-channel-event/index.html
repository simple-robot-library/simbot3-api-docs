QGChannelEvent

sealed class QGChannelEvent : QGBotEvent<EventChannel> (source)

子频道相关的事件。

QGChannelEvent 主要用于为子类型提供一个基础类型，其本身没有过多的事件类型约束。

子频道的增减视为频道服务器变更事件 OrganizationChangeEvent, 而子频道本身的修改事件则视为子频道变更事件 ChannelEvent + ChangeEvent。
