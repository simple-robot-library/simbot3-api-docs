messageFromReference

@ExperimentalQGApi
open suspend override fun messageFromReference(reference: MessageReference): QGMessageContent(source)

根据引用信息 reference 查询指定的文字子频道消息正文。 reference 必须是 QGReference 类型，并且不能缺失 QGReference.channelId， 因为查询消息内容除了 messageId 以外还需要 channelId 信息。

Throws

UnsupportedOperationException

如果 reference 类型不是 QGReference

IllegalArgumentException

如果 reference 的 channelId 为 null
