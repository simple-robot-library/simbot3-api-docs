setAdmin

abstract suspend fun setAdmin(enable: Boolean)(source)

设置当前群成员为管理员/撤销其管理员。

会根据 enable 在请求成功后影响 role 的值， 但是仅会影响 当前对象 内的属性值。

setAdmin 不保证并发安全也不会加锁， 如果并发请求 setAdmin，无法保证 role 的最终结果。

Parameters

enable

为 true 则为设置管理，false 则为取消管理。
