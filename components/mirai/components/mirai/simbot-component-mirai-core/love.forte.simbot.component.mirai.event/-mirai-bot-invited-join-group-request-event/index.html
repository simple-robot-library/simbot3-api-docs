MiraiBotInvitedJoinGroupRequestEvent

interface MiraiBotInvitedJoinGroupRequestEvent : MiraiSimbotBotEvent<BotInvitedJoinGroupRequestEvent> , GroupJoinRequestEvent(source)

Bot 被邀请加入一个群事件。

MiraiBotInvitedJoinGroupRequestEvent 在含义上类似于 MiraiGroupBotEvent, 但是 OriginalMiraiBotInvitedJoinGroupRequestEvent 不属于 OriginalMiraiGroupEvent, 因此当前事件类型不属于 MiraiGroupBotEvent.
