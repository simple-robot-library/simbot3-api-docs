MiraiMember

interface MiraiMember : GroupMember, MiraiContact, DeleteSupport, MiraiUserProfileQueryable(source)

一个由simbot包装为 GroupMember 的 OriginalMiraiMember 对象。

DeleteSupport

一个 mirai 的群成员是 支持删除 操作的, delete 行为相当于 踢出 操作。

当 originalContact 的类型不是 OriginalMiraiNormalMember 的时候，delete 行为将会无效。
