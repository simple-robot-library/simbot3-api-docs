InvalidMiraiRecallMessageCacheStrategy

object InvalidMiraiRecallMessageCacheStrategy : StandardMiraiRecallMessageCacheStrategy(source)

MiraiRecallMessageCacheStrategy 的最简实现，无效的缓存策略，即不进行缓存。

InvalidMiraiRecallMessageCacheStrategy 理所当然的是处理最快的缓存策略， 但使用后无法再从撤回事件中得到 撤回的消息内容。

InvalidMiraiRecallMessageCacheStrategy 是 MiraiBotConfiguration.recallCacheStrategy 的默认策略。
