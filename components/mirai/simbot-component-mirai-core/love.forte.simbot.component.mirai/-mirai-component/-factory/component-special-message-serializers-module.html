componentSpecialMessageSerializersModule

@JvmStatic
val componentSpecialMessageSerializersModule: SerializersModule(source)

当前组件中所提供的所有额外消息类型的序列化模块。

componentSpecialMessageSerializersModule 只包含组件内的消息序列化模块，而不包含mirai提供的消息序列化模块。 如果想要保证序列化不会出现问题，你可能需要使用 messageSerializersModule.
