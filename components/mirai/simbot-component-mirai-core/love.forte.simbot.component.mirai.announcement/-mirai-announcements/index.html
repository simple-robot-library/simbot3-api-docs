MiraiAnnouncements

@MiraiMappingType(target = Announcements::class)
interface MiraiAnnouncements(source)

表示一个群的公告列表 (管理器)，相当于 mirai Announcements。

API大多与 mirai Announcements 相对应，部分参数和类型会被替换为simbot或组件中的类型。
