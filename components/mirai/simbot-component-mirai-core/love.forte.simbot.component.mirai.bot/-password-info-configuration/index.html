通过 MiraiBotVerifyInfoConfiguration.passwordInfo 来提供多种形式的密码配置。

目前支持的形式有：

明文密码:

  • 普通明文密码

  • 环境变量明文密码

md5密码:

  • md5字符串密码

  • 环境变量md5字符串密码

  • md5字节组密码
