MiraiStrangerMessagePostSendEvent

interface MiraiStrangerMessagePostSendEvent : MiraiMessagePostSendEvent<Stranger, StrangerMessagePostSendEvent> , UserInfoContainer, MessageEvent, RemoteMessageContainer(source)

陌生人消息发送后的消息事件。此事件不会实现 ContactMessageEvent, 取而代之的是使用 UserInfoContainer (Stranger) , MessageEvent. 因此此消息本质上并非"陌生人发来的消息"，而只是对bot的行为的后置处理。
