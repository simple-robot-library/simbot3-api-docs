MiraiFriendMessagePostSendEvent

interface MiraiFriendMessagePostSendEvent : MiraiMessagePostSendEvent<Friend, FriendMessagePostSendEvent> , FriendInfoContainer, MessageEvent, RemoteMessageContainer(source)

好友消息发送后的消息事件。此事件不会实现 FriendMessageEvent, 取而代之的是使用 FriendInfoContainer, MessageEvent. 因此此消息本质上并非"好友发来的消息"，而只是对bot的行为的后置处理。
