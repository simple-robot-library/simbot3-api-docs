RequestMemberInviterInfo

data class RequestMemberInviterInfo(val id: LongID, val member: MiraiMember?) : MemberInfo(source)

MiraiMemberJoinRequestEvent 中的 requester 等相关属性的返回值。

如果在尝试获取此邀请人的时候，此人已经离开了群，那么 member 便可能为null。

当 member 为null的时候，只能获取到 id, 而 nickname 和 username 只会得到空字符串。
