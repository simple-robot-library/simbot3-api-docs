MiraiGroupSettingEvent

@BaseEvent
interface MiraiGroupSettingEvent<T, E : GroupSettingChangeEvent<T>> : MiraiSimbotBotEvent<E> , ChangedEvent(source)

与mirai的 群设置 相关的事件总类。

这些与设置变更相关的事件均属于 已变化事件, 事件源为群， 变化前后为对应设置项值。
