MiraiBotLeaveEvent

interface MiraiBotLeaveEvent : MiraiGroupBotEvent<BotLeaveEvent> , MemberDecreaseEvent(source)

机器人被踢出群或在其他客户端主动退出一个群. 在事件广播前 Bot.groups 就已删除这个群.

此事件属于一个 群成员减少事件, 减少的这个成员便是bot自己。
