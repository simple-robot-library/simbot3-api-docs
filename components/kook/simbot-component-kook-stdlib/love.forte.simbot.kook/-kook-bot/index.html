KookBot

interface KookBot : CoroutineScope, LoggerContainer, KookApiRequestor(source)

一个 KOOK BOT.

KookBot 承载了通过 WebSocket 的方式与KOOK服务器连接并订阅事件的能力， 以及通过各种 KOOK HTTP API 进行功能交互的能力。
