@Serializable
@SerialName(value = "header")
data class Header(val text: CardElement.Text) : CardModule(source)

标题模块

标题模块只能支持展示标准文本（text），突出标题样式。

说明：

  • text 为文字元素且 content 不能超过 100 个字

e.g.

{
    "type": "header",
    "text" : {
        "type": "plain-text",
        "content": "标题1"
    }
}
