图片

作用说明： 显示图片。

规则：

  • 图片类型（MimeType）限制说明：目前仅支持image/jpeg, image/gif, image/png这 3 种

  • 图片的 size 默认为 lg

e.g.

{
    "type": "image",
    "src" : "",
    "alt" : "",
    "size" : "sm|lg", // size只用在图文混排  图片组大小固定
    "circle" : true|false,
}
