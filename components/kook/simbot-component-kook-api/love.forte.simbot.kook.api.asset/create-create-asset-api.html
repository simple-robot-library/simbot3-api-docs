提供文件的 File 作为上传API。

文件在上传时会通过 File.readChannel 转化为 ByteReadChannel。

File 不会被立即读取，而是在实际发起请求时被读取。 API 进行请求时，可能会因 File 而导致产生各种异常，比如 IOException。
