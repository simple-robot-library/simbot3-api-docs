KookApiRequest

abstract class KookApiRequest<T> : API<KookApiRequestor, T> (source)

代表、包装了一个 Kook api的请求。

反序列化

KookApiRequest 面向 ktor, 并基于 kotlinx.serialization 进行反序列化。

Url内容

最终进行请求的 url 中部分参数（例如host）来自于 Kook.

不可变

此接口的实现类应当是不可变、可复用的。
