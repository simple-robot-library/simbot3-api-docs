class UnknownExtra : EventExtra

当一个事件反序列化失败的时候，会被尝试使用 UnknownExtra 作为 extra 的序列化目标。 如果是因为一个未知的事件导致的这次失败，则 UnknownExtra 便会反序列化成功并被推送。
