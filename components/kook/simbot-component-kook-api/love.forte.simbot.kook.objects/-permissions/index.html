权限

权限是一个unsigned int值，由比特位代表是否拥有对应的权限。 权限值与对应比特位进行按位与操作，判断是否拥有该权限。

判断是否有某权限

其中: permissions代表权限值，bitValue代表某权限比特位，1 << bitValue 代表某权限值。 permissions & (1 << bitValue) == (1 << bitValue);
