KookApiRequestedReceipt

class KookApiRequestedReceipt(val result: Any?, val isDirect: Boolean, val bot: KookComponentBot) : SingleKookMessageReceipt(source)

消息发送后的回执。

与 KookMessageCreatedReceipt 不同的是， KookApiRequestedReceipt 很可能并不是通过执行的消息api， 例如通过 KookRequestMessage 执行了一个任意地请求。

也正因此，此回执不支持 删除 操作。当使用 delete 时将会恒返回 false.
