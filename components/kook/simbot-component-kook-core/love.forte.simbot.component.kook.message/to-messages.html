toMessages

fun Event<Event.Extra.Text>.toMessages(): Messages(source)

将消息事件相关内容转化为 Messages.

Event.Extra转化规则
TextEventExtra将 Event.content 转化为 Text
ImageEventExtra将 ImageEventExtra.attachments 转化为 KookAttachmentMessage
FileEventExtra将 FileEventExtra.attachments 转化为 KookAttachmentMessage
VideoEventExtra将 VideoEventExtra.attachments 转化为 KookAttachmentMessage
CardEventExtra将 Event.content 转化为 Text
KMarkdownEventExtra将 Event.content 转化为 Text（会对特殊内容进行适当的移除）；
追加其他由 mention、mentionRoles、isMentionAll、isMentionHere 等信息而解析出来的 At、AtAll、KookAtAllHere 等信息。其中，AtAll、KookAtAllHere 至多只会各自出现一次。
其他将 Event.content 转化为 Text
