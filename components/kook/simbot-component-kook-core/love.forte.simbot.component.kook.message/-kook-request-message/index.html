KookRequestMessage

@KookSendOnlyMessage
data class KookRequestMessage(val request: KookApiRequest<*>) : KookMessageElement<KookRequestMessage> (source)

提供一个 KookApiRequest 作为原始的消息发送请求（例如 MessageCreateRequest）。

此消息会直接使用 request 作为消息发送的请求。

这是一个仅用于发送的消息，且不支持序列化。
