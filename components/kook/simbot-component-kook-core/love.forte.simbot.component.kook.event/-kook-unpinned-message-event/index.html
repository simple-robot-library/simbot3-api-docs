KookUnpinnedMessageEvent

abstract class KookUnpinnedMessageEvent : KookMessagePinEvent<UnpinnedMessageExtraBody> (source)

消息取消置顶事件。此事件的body也属于 ChannelEventExtraBody 类型的自类型， 代表一个新的消息被设置为了目标频道的置顶消息。

此事件为 ChangedEvent, 事件源为发生事件的频道服务器，变动主体为被设置为置顶消息 的ID。 由于事件无法确定变更前的消息，因此只能获取到 删除前的 消息ID，而 after 恒为null。
