simbot-component-tencent-guild-api

对QQ频道API的最基本封装，不包含额外的功能性实现（比如bot、事件等）。

如果希望使用一个仅仅是针对API的原始封装模块而不需要基础/高级的事件流程控制，那么请考虑使用api模块。
