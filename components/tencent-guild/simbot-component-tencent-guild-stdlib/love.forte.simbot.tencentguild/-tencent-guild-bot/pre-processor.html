preProcessor

abstract fun preProcessor(processor: suspend Signal.Dispatch.(decoder: Json, decoded: () -> Any) -> Unit)(source)

添加一个事件预处理器。 processor 中的参数 decoded 所代表的为 decoder.decodeFromJsonElement(eventType.decoder, data) 后的结果， 但是为了避免在多个事件处理器中频繁进行反序列化，因此提供这个 decoded 来预先提供一个懒实例化的获取器。

预处理器与 processor 不同的是，preProcessor 所注册的所有事件会在接收到事件的时候以协程同步的方式按照顺序执行， 而后再交由下游的 processor 处理。
