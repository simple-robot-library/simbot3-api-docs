Simple Robot Component Tencent Guild
3.0.0.0-alpha.3

simbot-component-tencent-guild-core

基于core模块，基于simbot3核心库 对QQ频道API功能的完整性实现。 这是此仓库作为simbot组件库的主要 "证据" ，也是实际意义上的组件模块。

core模块尽可能大限度的以QQ频道的角度去实现simbot核心库中的API，是一个较高封装程度的模块。同样的，使用此模块可以使用绝大多数的simbot特性， 包括它的API风格以及Spring Boot Starter等特性。

有关simbot3的内容，可以前往其官方网站或仓库首页了解更多。

使用

Gradle Kotlin DSL

implementation("love.forte.simbot:simbot-core:$SIMBOT_VERSION") // 必须显式指定simbot相关依赖比如此核心库或spring-boot-starter
implementation("love.forte.simbot.component:simbot-component-tencent-guild-core:$VERSION")
