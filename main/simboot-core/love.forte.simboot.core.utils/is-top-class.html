isTopClass

@InternalSimbotApi
fun Class<*>.isTopClass(): Boolean(source)

判断一个 Class 是否 疑似为 一个Kotlin中的顶层函数。

当一个 Class 符合如下条件时得到 true。

  • Class.getConstructors 为空。

  • Class.getInterfaces 为空。

  • 此 class 存在 @kotlin.Metadata 注解。

  • 是 public、final 的

  • 不是抽象的。

  • 不是枚举、不是注解、不是数组。

Note: 此函数不捕获异常。
