sealed class Timestamp : Comparable<Timestamp> (source)

一个 时间戳 。

通常情况下，second 或 millisecond 得到-1的值，那么就说明此时间戳并不是一个真正的时间戳， 而是一个不被支持的默认值。 通过 Timestamp 你可以直接通过 isSupport 对支持情况进行判断。
