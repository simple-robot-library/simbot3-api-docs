Timestamp

@Serializable(with = TimestampSerializer::class)
sealed class Timestamp : Comparable<Timestamp> (source)

一个 时间戳 。

通常情况下，second 或 millisecond 得到-1的值，那么就说明此时间戳并不是一个真正的时间戳， 而是一个不被支持的默认值。 通过 Timestamp 你可以直接通过 isSupport 对支持情况进行判断。

简单包装器

Timestamp 是一种简单包装器类型，因此建议那些对外公开的 Timestamp 属性使用 即用即造 的形式，即不会急切的初始化属性， 而是每次获取时临时创建。

例如：

val container = ...

// 下面会产生3个 Timestamp 对象
useTimestamp(container.timestamp)
useTimestamp(container.timestamp)
useTimestamp(container.timestamp)

useTimestamp(timestamp)
