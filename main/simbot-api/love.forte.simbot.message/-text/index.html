Text

@Serializable
open class Text(val text: String) : BaseStandardMessage<Text> , PlainText<Text> (source)

一个文本消息 Text。Text 是 PlainText 基础实现类型。

文本消息可以存在多个，但是对于不同平台来讲，有可能存在差异。 部分平台会按照正常的方式顺序排列消息，而有的则会组合消息列表中的所有文本消息为一个整体。
