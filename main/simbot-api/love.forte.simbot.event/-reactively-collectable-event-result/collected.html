abstract fun collected(collectedContent: Any?): EventResult(source)

当响应式结果 content 被收集完毕后通过 collected 提供其收集的结果 collectedContent, 并作为一个新的 EventResult 结果提供给 EventProcessingResult.resultsView.

collected 的结果不会再被二次收集, 因此假若 collectedContent 仍然为响应式类型, 则它们将会被忽略并直接作为结果返回.
