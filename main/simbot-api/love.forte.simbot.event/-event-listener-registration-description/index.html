EventListenerRegistrationDescription

abstract class EventListenerRegistrationDescription(source)

用于向 EventListenerRegistrar 中注册监听函数的信息描述. 可以在 EventListener 之外提供更多可能有效的附加信息.

除了 listener 以外, 其他的所有属性信息均不保证一定会被 EventListenerRegistrar 的具体实现所处理, 对于它们是否能够被应用, 需要查阅具体实现的说明.
