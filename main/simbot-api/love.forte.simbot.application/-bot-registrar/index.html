interface BotRegistrar(source)

应用于 ApplicationBuilder.bots 中的bot注册函数, 提供一个通过 BotVerifyInfo 注册的通用bot注册函数。

BotRegistrar 会通过 BotVerifyInfo 中的 组件id 去当前环境中寻找对应组件的、实现了 Bot注册器 的 事件提供者, 并尝试注册此bot。
