StageLoop

expect open class StageLoop<S : Stage<S>>(source)

可挂起的事件循环器。

StageLoop 内部持有可变状态，一个 StageLoop 实例建议同时只进行一个循环任务。

sealed FooStage : Stage<FooStage>() {
    object Start : FooStage() { ... }
    // ...
}

val stageLoop = ...
stageLoop.appendStage(FooStage.Start)

stageLoop.loop()
