fun interface InSession<T, R>(source)

使用于 ContinuousSessionContext.session 中的 receiver 逻辑函数。

在 Java 中，可以使用 InSessions 中提供的各种静态工厂函数构建它，例如 InSessions.async、InSessions.mono 等。

在 ContinuousSession 中使用时，我们强烈建议使用非阻塞的 InSession 实现， 或者为 ContinuousSession 的调度器配置为 虚拟线程调度器 。

var dispatcher = ExecutorsKt.from(Executors.newVirtualThreadPerTaskExecutor());
