SuspendReserve

interface SuspendReserve<out T>(source)

一个用于表示“预备”挂起函数的类型， 用于提供一层非挂起的“中转”转化器。

Java 中提供部分默认可应用于 transform 的转化器，可参考 SuspendReserves 中提供的各项静态工厂。

e.g.:

foo.runReserve().transform(SuspendReserves.mono()); // 将结果转化为 reactor 中的 Mono<T>
