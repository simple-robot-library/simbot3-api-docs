由平台实现的 MutableMap computeIfPresent 操作。

提供 key 从 map 中检索匹配的值，如果没有与之匹配的值， 则通过 remapping 计算并存入后返回此计算值，否则直接返回得到的匹配值。

在 JVM 平台中，会被委托给 java.util.Map.computeIfPresent， 其他平台会有相应的实现，但是可能无法保证原子操作。
