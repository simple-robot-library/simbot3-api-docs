simbot-logger

simbot3的多平台日志实现，为其它模块提供统一的日志API。

在JVM中，日志API将会委托予 slf4j-api。

在JS中，将会使用console API。

而在native目标中，将会通过Kotlin基于最简单的控制台输出实现。

平台支持

simbot-logger 是基于 Kotlin Multiplatform 的多平台模块，其支持所有可用目标，包括JVM、JS、Native。

JVM

JVM 平台中依赖并实现 slf4j-api，但是不提供 slf4j-api 的实现。因此在JVM平台中如果只引用 simbot-logger 模块则无效果， 需要额外引用其他的 slf4j-api 实现，或者使用 ../simbot-logger-slf4j-impl。

JS

JS 平台中基于 console API 实现日志输出。

Native

Native 平台通过简单的标准输出 (println) 实现日志输出。
