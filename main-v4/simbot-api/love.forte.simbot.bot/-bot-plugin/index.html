BotPlugin

interface BotPlugin : Plugin(source)

一个与 Bot 相关的插件。是一个具有特殊意义的插件类型。

BotPlugin 本身不提供任何API，请参考 BotManager 或 AutoConfigurableBotPlugin。 实现者通常会提供个性化的注册API来支持注册能力。

如果希望 BotPlugin 能支持在 quantcat 模块下多组件环境中自动解析配置文件并加载， 则需要考虑实现 AutoConfigurableBotPlugin，仅实现 BotPlugin 无法支持自动注册 Bot。
