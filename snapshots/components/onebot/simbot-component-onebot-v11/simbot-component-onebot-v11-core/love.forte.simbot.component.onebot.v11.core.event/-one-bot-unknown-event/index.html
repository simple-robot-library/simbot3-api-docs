OneBotUnknownEvent

@FragileSimbotAPI
data class OneBotUnknownEvent(val sourceEventRaw: String?, val sourceEvent: UnknownEvent) : OneBotEvent(source)

OneBot组件对一个未知事件 UnknownEvent 的包装。

与 OneBotUnsupportedEvent 不同的是， OneBotUnknownEvent 的事件类型明确为 UnknownEvent， 它是在 OneBot 协议本身上的“未知”，也就是指无法解析事件的报文。
