OneBotMessageSegment

interface OneBotMessageSegment(source)

OneBot11的 消息段 类型定义。 其中，消息段的类型 type 由多态序列化类型名决定，而 data 的类型则由实现者决定。

定义的可序列化子类型会被统一以 OneBotMessageSegment 的多态类型被添加到 OneBot11Component.serializersModule 中。 (通过 includeAllOneBotSegmentImpls )

Message Element

可以通过 OneBotMessageSegmentElement 将 OneBotMessageSegment 包装为 Message.Element 的子类型。
