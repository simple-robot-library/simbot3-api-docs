UnknownEvent

@FragileSimbotAPI
data class UnknownEvent(val time: Long, val selfId: LongID, val postType: String, val raw: String) : Event(source)

用于“兜底”的 Event 类型实现。 当出现了尚未支持或某种未知的事件体，无法对应到任何现有的已定义结构时， 则应当将其解析并包装为 UnknownEvent。

FragileAPI

这是一个具有特殊规则的事件类型。 随着版本地更新，UnknownEvent 中可能出现的事件类型会越来越少。 因此此类型只适用于“兜底”，不应过度依赖。
