MiraiGroupTempMessagePostSendEvent

interface MiraiGroupTempMessagePostSendEvent : MiraiMessagePostSendEvent<Member, GroupTempMessagePostSendEvent> , GroupInfoContainer, MemberInfoContainer, MessageEvent(source)

群临时会话消息发送后的消息事件。此事件不会实现 ContactMessageEvent, 取而代之的是使用 GroupInfoContainer, MemberInfoContainer, MessageEvent. 此消息本质上并非"群成员的临时消息"，而只是对bot的行为的后置处理。
