Paragraph

@Serializable
@SerialName(value = "paragraph")
data class Paragraph(val cols: Int, val fields: List<CardElement.Text>) : CardElement(source)

结构体 的 区域文本。

作用说明： 支持分栏结构，将模块分为左右两栏，根据顺序自动排列，支持更自由的文字排版模式，提高可维护性

e.g.

{
    "type": "paragraph",
    "cols": 3, //移动端忽略该参数
    "fields" : [
    ],
}

规则：

  • cols 是 int,可以的取值为 1-3

  • fields 可以的元素为 text 或 kmarkdown.

  • paragraph 最多有 50 个元素
