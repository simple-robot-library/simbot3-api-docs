KookMemberExitedChannelEvent

abstract class KookMemberExitedChannelEvent : KookMemberChannelChangedEvent<UserExitedChannelEventBody> , MemberDecreaseEvent(source)

Kook 成员离开(频道)事件。

此事件被触发时，相关成员已经被移除自对应频道，且终止了内置的所有任务。 因此 before 不可用于执行禁言等操作，before 也不会存在于当前频道成员中。
