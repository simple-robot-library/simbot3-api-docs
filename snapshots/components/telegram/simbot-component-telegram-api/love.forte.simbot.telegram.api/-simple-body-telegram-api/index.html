SimpleBodyTelegramApi

abstract class SimpleBodyTelegramApi<R : Any> : TelegramApi<R> (source)

An ordinary extensible TelegramApi. Post is used if body is not null, application/json is used if body is not MultiPartFormDataContent, and application/x-www-form-urlencoded is used otherwise.

If body is OutgoingContent other than MultiPartFormDataContent, remember to override headers to provide the necessary information as needed.
