interface QGChannel : Channel, CoroutineScope, QGObjectiveContainer<Channel> (source)

QQ频道中的子频道。

关于 Organization 实现

实际上 Channel 类型并不应该实现 Organization 这个接口。 在子频道中，所有从 Organization 继承而来的属性大部分均不可用，例如 maximumMember 等。 这属于 simbot v3 的设计缺陷之一。
