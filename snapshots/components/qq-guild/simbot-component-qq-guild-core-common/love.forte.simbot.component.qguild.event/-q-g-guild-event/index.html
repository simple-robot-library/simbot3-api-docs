QGGuildEvent

sealed class QGGuildEvent : QGEvent<EventGuild> (source)

频道变更事件相关。

QGGuildEvent 主要为子类型提供基本类型，但 QGGuildEvent 本身没有过多事件类型约束。

频道新增事件 表示bot进入到了一个新的频道中。

频道更新事件 表示某个频道的信息发生了更新。

频道移除事件 表示bot离开了某个频道。 此事件与前两者有较大差别：它不实现 GuildEvent —— 毕竟频道已经离开（不存在）了。
