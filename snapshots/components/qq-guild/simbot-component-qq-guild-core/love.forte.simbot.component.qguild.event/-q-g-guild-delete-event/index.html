QGGuildDeleteEvent

abstract class QGGuildDeleteEvent : QGGuildEvent(source)

频道删除事件。

QGGuildDeleteEvent 不实现 GuildEvent，因为事件触发时已经离开频道，不存在可稳定获取的 QGGuild 实例。 但是此事件额外提供属性 guild, 代表获取一个可能存在的不稳定 QGGuild. 详细内容参考 guild 说明。

触发时机：

  • 频道被解散

  • 机器人被移除
