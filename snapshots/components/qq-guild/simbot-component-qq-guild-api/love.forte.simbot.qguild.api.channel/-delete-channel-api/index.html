class DeleteChannelApi : QQGuildApi<Unit> (source)

删除子频道

用于删除 channel_id 指定的子频道。

  • 要求操作人具有 管理子频道 的权限，如果是机器人，则需要将机器人设置为管理员。

  • 修改成功后，会触发子频道删除事件。

注意

子频道的删除是无法撤回的，一旦删除，将无法恢复。
