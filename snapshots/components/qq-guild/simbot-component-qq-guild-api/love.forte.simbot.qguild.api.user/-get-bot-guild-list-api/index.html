class GetBotGuildListApi : GetQQGuildApi<List<SimpleGuild>> (source)

获取用户频道列表

用于获取当前用户（机器人）所加入的频道列表，支持分页。

当 HTTP Authorization 中填入 Bot Token 是获取机器人的数据，填入 Bearer Token 则获取用户的数据。
