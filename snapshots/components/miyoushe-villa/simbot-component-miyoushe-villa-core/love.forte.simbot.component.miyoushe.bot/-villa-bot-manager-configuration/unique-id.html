uniqueId

var uniqueId: Boolean(source)

是否要求相同 id 的bot同时只能存在一个。

如果为 false 则允许注册多个相同 id 的 VillaBot。 此时如果通过 VillaBotManager.get 获取则会得到第一个匹配的结果（通常来讲是最早注册的那个）。

如果为 true（默认）则注册时如果已经存在相同 id 的 bot，抛出异常。
