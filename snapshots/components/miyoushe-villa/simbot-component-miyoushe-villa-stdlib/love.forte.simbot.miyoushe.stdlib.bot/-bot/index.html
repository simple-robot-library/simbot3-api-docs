Bot

interface Bot : CoroutineScope(source)

一个米游社大别野的 bot。

Bot 基于 Websocket 与米游社大别野建立连接并订阅、处理事件。

终止

Bot 可能会通过 logout、cancel 主动终止，或收到了 PKickOff 数据包而被动终止。 Bot 被终止后 不可重新启动。
