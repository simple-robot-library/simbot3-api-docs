心跳请求命令字

心跳检查

登录成功后，需要每隔一段时间发送一次 HeartBeatRequest 到服务端，保活长连接。

服务端超过一分钟没有从长连接读取到新数据，就会将连接标记失活，最终断开连接。

简单实现，建议每20秒一次心跳包，更好的做法是结合其它数据包判断，距离上一个数据包发出过了20秒就发出一个心跳包。
