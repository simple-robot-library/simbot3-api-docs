AuditContentType

enum AuditContentType : Enum<AuditContentType> (source)

送审内容的类型

支持类型：文本、图片 如果送审内容为文本，则 content_type=AuditContentTypeText, 将送审内容填充到 audit_content 字段中。

如果送审内容是图片，则 content_type=AuditContentTypeImage，将图片url填充到 audit_content 字段中。

注：

  • 如果content_type传入值为空，默认当作文本类型处理

  • 如果送审图片，需先调用转存接口，将转存后的URL填充到audit_content中
