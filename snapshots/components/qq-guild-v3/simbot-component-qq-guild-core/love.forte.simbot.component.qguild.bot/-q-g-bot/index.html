QGBot

interface QGBot : Bot(source)

一个 QQ频道Bot 的 simbot组件实现接口， 是一个具有功能的 QQ频道组件Bot。

AppID 与 用户ID

QGBot 的 id 是用于连接服务器的 appId 而并非系统用户ID。 如果希望获取bot的 用户ID, 至少执行过一次 QGBot.start 后通过 userId 获取， 或者通过 me 实时查询。

Note: QGBot仅由内部实现，对外不稳定
