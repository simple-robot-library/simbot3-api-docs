Cancelled

data object Cancelled : ApplicationLaunchStage<NormalApplicationEventHandler<Application>> (source)

在 ApplicationLauncher 被关闭后。 此时的 Application 已经被关闭，但是 Application.cancel 尚未返回时。 此时的 Application.isActive 将返回 false，Application.join 也不会再被挂起。

Cancelled 中的操作应当迅速且安全，避免抛出异常，也不应长时间阻塞。

但是这些行为都是不推荐的。 这可能会导致无法预知的问题进而引发灾难。
