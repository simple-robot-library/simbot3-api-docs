fun interface TypedJBlockEventListener<E : Event>(source)

一个事件 Event 的阻塞监听器。也可以称之为事件处理器。

是针对JVM平台的兼容类型，可以通过 toListener 转化为 EventListener 类型。

会针对指定的类型进行事件处理。如果类型不匹配则会返回 EventResult.invalid。
