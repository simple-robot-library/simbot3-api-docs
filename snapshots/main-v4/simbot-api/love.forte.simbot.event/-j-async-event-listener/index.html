JAsyncEventListener

fun interface JAsyncEventListener(source)

一个事件 Event 的异步监听器。也可以称之为事件处理器。

是针对JVM平台的兼容类型，可以通过 toListener 转化为 EventListener 类型。

如果希望针对一个具体的类型进行处理，可参考 TypedJAsyncEventListener。
