一个艾特消息。

是针对“提及”的常见标准表现形式。 默认情况下 At 表现为针对某个用户的提及（即默认的 type 为 "user"）。

当同一个组件中可能存在多种类型的提及时，可以选择通过约定不同的 type 来区分， 也可以选择实现更多扩展消息元素类型来做区分。
