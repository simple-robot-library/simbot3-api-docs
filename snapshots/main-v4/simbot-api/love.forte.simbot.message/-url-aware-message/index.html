interface UrlAwareMessage(source)

表示为一个可以得知 URL 地址的消息元素， 例如 UrlAwareImage。 应当由某个 Message.Element 的子类型实现。

UrlAwareMessage 主要用于那些从服务端返回的消息元素使用， 而不是本地构建的消息元素。

abstract suspend fun url(): String

获取到链接字符串。 如果链接信息包含在响应数据中，则会立即返回， 否则会挂起并查询链接信息（例如通过网络接口查询）。 如果需要查询，其内部不会缓存结果，因此每次调用 url 均会产生挂起与查询行为。
