val ifNullPass: Boolean = false

当 value 对消息事件进行匹配时， 如果消息的 纯文本内容 为 null， 是否直接放行。如果为 true, 则纯文本内容为 null 的时候视为匹配通过，反之则为匹配失败。默认为 false。 此参数只有当 value 不为空的时候有效。
