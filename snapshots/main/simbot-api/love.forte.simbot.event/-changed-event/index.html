ChangedEvent

@BaseEvent
interface ChangedEvent : ChangeEvent(source)

一个 变更 事件，代表一个已经变更结束的事件。 ChangedEvent 是 ChangeEvent 事件的 事实 描述， 一般来讲代表了一个已经发生变更的事件。

source 仍然为变更内容的载体，before 代表一个变更之前的瞬时状态， after 则代表为一个变更后的状态。
