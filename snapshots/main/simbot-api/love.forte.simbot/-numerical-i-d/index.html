NumericalID

@SerialName(value = "ID.Number")
@Serializable
sealed class NumericalID<N : Number> : ID, NumberSimilarly(source)

以一个 数字 作为字面值的 ID.

这个数字可能是 Int, Long, Float, Double,

或者一个平台下相关的 其他 Number 实现。

示例:

// Kotlin
val id: LongID = 123L.ID
