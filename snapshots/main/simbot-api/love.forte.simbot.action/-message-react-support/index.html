interface MessageReactSupport(source)

消息回应支持。

此接口通常标记在一个 消息事件 上，代表这个消息能够被 回应。

很多允许频道的平台都会有这个功能，能够标记的内容也应当属于一个 Message, 不过它们大多数的时候支持的类型有限， 例如 emoji 或者一些自定义表情。

MessageReactSupport.react 将不会对参数 message 进行类型限制，相对的，实现者需要对参数消息进行校验，并在存在不匹配的情况时抛出异常。

回应 与 回复 不同， 回复常常类似于针对某个消息而回复一条消息，会产生一条新的消息； 而回应则更多的是对于一个消息"作出回应"，通产情况下不会产生新的消息，一般会表现为标记一个表情。
