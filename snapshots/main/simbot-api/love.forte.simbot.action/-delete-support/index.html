interface DeleteSupport(source)

允许一种删除行为。 标记一个消息为可删除的。

对于一种删除行为来讲，它最常见的含义就是撤回（针对于消息, 参考 RemoteMessageContent、MessageReceipt） 和 踢出/移除（针对于好友、群成员等, 但是没有提供默认实现）。
