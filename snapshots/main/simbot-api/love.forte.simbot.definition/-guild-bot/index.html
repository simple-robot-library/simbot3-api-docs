GuildBot

interface GuildBot : OrganizationBot(source)

一个 Bot 在一个 频道服务器 中所扮演的对象。

GuildBot 实现 OrganizationBot, 并对外提供转化为 Member 的能力, 以允许将自身转化为在目标频道服务器中的成员。
