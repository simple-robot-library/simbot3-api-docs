abstract override fun close()(source)

StandardResource 在使用的过程中可能会产生一些需要手动进行 close 的产物， 因此在不使用 StandardResource 的时候，使用 close 对其进行关闭。

close 操作仅代表针对当前的 Resource 对象本身，不会对任何已经产生的结果造成影响。
