fun match(matcher: suspend EventListenerProcessingContext.(E) -> Boolean)(source)

配置当前监听函数的匹配函数。

listen(FooEvent) {
   match { condition } // return Boolean
   handle { ... }
}

match 函数允许多次使用。当执行多次 match 时，其效果相当于每次配置的条件之间通过与(&&)相连接。

例如：

listen(FooEvent) {
   match { condition1 }
   match { condition2 }
   match { condition3 }

   handle { ... }
}
Content copied to clipboard

其效果等同于：

listen(FooEvent) {
   match { condition1 && condition2 && condition3 }

   handle { ... }
}
