SimpleScope

object SimpleScope(source)

应用于 EventProcessingContext.getAttribute 中，提供部分标准作用域。

simbot所提供的默认 EventProcessingContext 实现会支持 SimpleScope 中的作用域，并且建议第三方实现也针对 SimpleScope 提供支持。
